# Video Converter

A electron desktop app to convert videos.

![video-converter](https://user-images.githubusercontent.com/11765228/41272573-43163602-6e48-11e8-9e43-e79a93bfedd3.png)

## Getting Started

`npm install`

Start dev server:

`npm start`

In a new terminal window:

`npm run electron`
